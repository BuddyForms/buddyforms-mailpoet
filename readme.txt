Let your users subscribe to MailPoet lists from BuddForms

== Description ==

With this integration, your website visitor and current users can be added to an existing list of MailPoet when they submit a Form.

##Requirement
This extension need BuddyForms Professional and MailPoet. If you need more than one extension we recommend to take a look to the [ThemeKraft Bundle](https://themekraft.com/products/themekraft-bundle/)

##Features
- Add a logged-in user to a list
- Remove a logged-in user from a list
- Add logged-out  user to a list
- Let your user select one or many lists to be subscribed
- Force a user to be added to a list
- Use the User Form Elements to update the Name and Last Name of an existing subscriber

##Use-cases
- Force a user to belong to a specific list(s) 
- Let the user add/remove by itself to existing MailPoet lists

##Source
